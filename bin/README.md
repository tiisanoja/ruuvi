# This folder contains compiled binaries

Descption of binaries and scripts

| Command | Description |
| --- | --- |
| bin/ruuvi | binary for the CPU architecture in which build was made | 
| bin/ruuvi.arm64 | binary for arm 64-bit CPU architecture. It can be run on Rasberry PI. |
| startRuuvi.sh | Bash script to start application. Note: It ill start bin/ruuvi and not bin/ruuvi.arm64 |
